---

## Run Locally

```bash
# Clone the repo
git clone https://github.com/faizanmakheel/portfolio.git

# Open in browser
open portfolio.html
```

That's it.

---

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to the `main` branch, `/ (root)` folder
4. GitHub will publish it at `https://faizanmakheel.github.io/portfolio`

> Update the live site link at the top of this README once Pages is active.
